## Step 1: Upload the license in Studio[​](https://www.rasa.com/docs/studio/installation/setup-guides/license-activation/#step-1-upload-the-license-in-studio 'Direct link to Step 1: Upload the license in Studio')

In order to activate Studio.

1. Navigate to Studio web client and you will be prompted to upload a license file
 
 ![image](https://www.rasa.com/docs/assets/images/upload-license-a1f2ec1714b7ee3b26e0d7a6e7dd256a.png)
 
 ![image](https://www.rasa.com/docs/assets/images/select-license-24cf7f26e7189369cc501855c084a7e0.png)
 
2. Hit the `Get Started` button after the license upload to start working with Studio
 
 ![image](https://www.rasa.com/docs/assets/images/license-activated-29cab18778a4916dd6dc38cd056565cc.png)
