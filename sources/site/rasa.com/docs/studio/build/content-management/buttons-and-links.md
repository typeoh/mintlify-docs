# Source: https://rasa.com/docs/studio/build/content-management/buttons-and-links

The buttons and links section of the CMS is a centralised place for content managers and flow builders to create and manage all the buttons and links used in an assistant.

Access from Buttons and Links navigation item under CMS on the navigation menu.

![image](https://rasa.com/docs/assets/images/buttons-and-links-cms-f6a7a3cf865d2f966ddbe1e60e78f15e.png)

### How to Create a Button[​](https://rasa.com/docs/studio/build/content-management/buttons-and-links/#how-to-create-a-button 'Direct link to How to Create a Button')

1. Click on the “Create new” button on the top right of the screen, or on the middle of the screen if no buttons or links exist yet.

![image](https://rasa.com/docs/assets/images/buttons-and-links-01-6820ef8b5b80d7d1de0db415d7d81fc5.png)

2. Type in a name and select the button payload type, and then click the save button.

![image](https://rasa.com/docs/assets/images/buttons-and-links-02-8a6717081fcc3924c313eb00cde549e1.png)

### How to Create a Link[​](https://rasa.com/docs/studio/build/content-management/buttons-and-links/#how-to-create-a-link 'Direct link to How to Create a Link')

1. Click on the “Create new” button on the top right of the screen.

![image](https://rasa.com/docs/assets/images/buttons-and-links-03-27bc6d1fc0f0e65b9ebe1d52f131ce10.png)

2. Click the “Create link” radio button, and then type in the link title, fill in the link URL and click the save button.

![image](https://rasa.com/docs/assets/images/buttons-and-links-04-d5240588f6718978fc21c70f022b09ed.png)

### How to Manage Buttons and Links[​](https://rasa.com/docs/studio/build/content-management/buttons-and-links/#how-to-manage-buttons-and-links 'Direct link to How to Manage Buttons and Links')

1. Select the button or link from the list on the left

![image](https://rasa.com/docs/assets/images/buttons-and-links-05-2faac173a83db39bf8342fea72627cd7.png)

2. Click the “See responses” button to view which responses are using this button or link. You can also click on the response name to navigate to it on the Response CMS.

![image](https://rasa.com/docs/assets/images/buttons-and-links-06-52d77cff601c565932995f70a3f6991d.png) ![image](https://rasa.com/docs/assets/images/buttons-and-links-07-aaeb5255f159edcdfa41e2378df7816d.png)

3. Click the “Delete” button to delete the button from the assistant.

![image](https://rasa.com/docs/assets/images/buttons-and-links-08-abf25aef9f37ba1823494b1a75241fc0.png)

4. Change any button or link values or types using the fields on the page.

![image](https://rasa.com/docs/assets/images/buttons-and-links-09-18e918b899f97afa66aa5d59429dbbe9.png)
