## Requirements[​](https://rasa.com/docs/pro/installation/overview/#requirements 'Direct link to Requirements')

Regardless of installation method, you will need:

- **Rasa License Key** — request a free Developer Edition key [here](https://rasa.com/rasa-pro-developer-edition-license-key-request/). Read more about [licensing](https://rasa.com/docs/pro/installation/licensing/).
- **LLM Provider API Key** — not needed for the [tutorial](https://rasa.com/docs/pro/tutorial/), but required for your own projects. See [supported LLM providers](https://rasa.com/docs/reference/config/components/llm-configuration/).
