## How Rasa Works

Rasa assistants are powered by **CALM** (Conversational AI with Language Models) — a dialogue system that uses LLMs to understand user intent while following your predefined business logic. This means you get the fluency of a large language model with the reliability of deterministic workflows.

<Steps>
  <Step title="Get a license">
    Start with the free Developer Edition — valid for up to 1,000 conversations per month. No credit card required.
  </Step>
  <Step title="Install Rasa Pro">
    Install via `pip` or Docker, initialize a project with `rasa init`, and connect MCP tools to your IDE.
  </Step>
  <Step title="Build your assistant">
    Define flows in YAML, write custom actions in Python, and configure your LLM provider of choice.
  </Step>
  <Step title="Test and deploy">
    Run end-to-end tests, deploy on Kubernetes or Docker, and monitor performance in production.
  </Step>
</Steps>
